#  Netflix Data Analysis using PostgreSQL

## Project Overview

This project analyzes the Netflix Movies & TV Shows dataset using PostgreSQL to answer real-world business questions using SQL.

## Dataset
Netflix Movies & TV Shows Dataset

## Tools Used
- PostgreSQL
- pgAdmin 4
- SQL

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- CASE Statements
- Common Table Expressions (CTEs)
- Window Functions
- String Functions
- Date Functions

## Business Questions Solved

- Movies vs TV Shows
- Total Netflix titles
- Movies released after 2020
- Most common ratings
- Top 5 countries with the highest content
- Longest movie
- Content added in the last 5 years
- Content by a specific director
- TV Shows with more than 5 seasons
- Most popular genres
- Year-wise Netflix content released in India
- Documentary movies
- Content without directors
- Salman Khan movies in the last 10 years
- Top actors in Indian Netflix content
- Categorizing content using CASE statements and CTEs

- Data Cleaning
- Data Exploration
- Business Problem Solving
- SQL Query Writing
- Analytical Thinking
